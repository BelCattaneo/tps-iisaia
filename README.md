# TPs - Introducción al Desarrollo de Software Asistido por IA

Trabajos prácticos de la materia **Introducción al Desarrollo de Software Asistido por IA** (CEIA).

## Contenido

### [`tp1/`](./tp1) — Tetris + UX

- **`index.html`** — Implementación de Tetris en HTML/JS, generada con asistencia de IA. Abrir directamente en el navegador.
- **`prompts.md`** — Chat completo con los prompts utilizados para obtener el HTML.
- **`TP UX prompts.pdf`** — Documentación de los prompts utilizados y el proceso de UX.

### [`tp2/`](./tp2) — OpenAPI para sistema de mapeo de puntos de venta

- **`openapi.yaml`** — Especificación OpenAPI 3.0.3 de una API con recursos `node` y `operation` (endpoints GET/POST/PATCH/DELETE), generada con asistencia de IA.
- **`prompts.md`** — Chat completo con los prompts utilizados para obtener la especificación.
